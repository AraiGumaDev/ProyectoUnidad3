Registro de consumo de Agua

Este es un programa que podría usar una entidad estilo EPM para ver las estádisticas y datos del consumo de los habitantes de una comunidad para así generar concientización. Este proyecto se alinea con el ODS 6 que busca que todas las personan puedan tener acceso a un minímo vital de agua y se le de un correcto uso. En la comuna en la que vivo últimamente hay denuncias sobre que el agua se esta usando de manera indevida para actividades como lavader carros, un programa como este se puede aplicar en cualquier comunidad y podría ayudar a identificar quienes son los individuos que hacen mal uso de este preciado recurso.
