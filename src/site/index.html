Registro de consumo de Agua

Este es un programa que podría usar una entidad estilo EPM para ver las estádisticas y datos del consumo de los habitantes de una comunidad para así generar concientización. Este proyecto se alinea con el ODS 6 (Agua limpia y saneamiento) ya que, busca que todas las personas puedan tener acceso a un minímo vital de agua y se le dé un correcto uso. Últimamente en una comunidad hay denuncias sobre que el agua se está usando de manera indebida para actividades como lavar vehículos particulares. Un programa como este se puede aplicar en cualquier comunidad y podría ayudar a identificar quiénes son los individuos que hacen mal uso de este preciado recurso.
